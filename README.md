# My First 2D Game 🎮

Welcome to my very first game development project! This is a basic 2D game built to understand the core concepts of the Unity Engine and C# scripting. 

## 🛠️ Tech Stack
* **Game Engine:** Unity (2D Core)
* **Language:** C#
* **IDE:** Visual Studio Code

## 💡 What I Learned
Transitioning from standard software programming to game development, this project helped me bridge the gap between traditional logic and interactive game loops. Key takeaways include:
* Understanding Unity's interface, GameObjects, and components.
* Applying Object-Oriented Programming principles to control player mechanics.
* Working with basic 2D physics (Rigidbodies and Colliders).
* Linking C# scripts to visual elements in the scene.

## 🚀 Future Scope
This is a foundational build. I plan to revisit and polish this project by adding:
* Interactive UI elements (Score, Main Menu).
* Audio and Sound Effects.
* More complex level design.
